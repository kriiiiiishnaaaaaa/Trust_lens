Quick Start Guide
# 1. Install dependencies
pip install -r requirements.txt

# 2. Configure environment
cp .env.example .env
# Edit .env: set DATABASE_URL, MODEL_PATH

# 3. (Optional) Fine-tune model on your dataset
python model/train.py --data_dir /path/to/data --epochs 20

# 4. Run the server
python app.py

# → Open http://localhost:5000
